# Reed Family Racing Starter Website

## Upload steps

1. Unzip this folder.
2. Upload all files to your GitHub repository.
3. In Netlify, import the GitHub repo.
4. Build command: leave blank.
5. Publish directory: `.`
6. Deploy the site.
7. In Netlify, enable Identity.
8. Enable Git Gateway.
9. Invite yourself as a user.
10. Go to `/admin` on your live site.
